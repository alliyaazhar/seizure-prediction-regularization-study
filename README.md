# Seizure Prediction Using Logistic Regression

## Objective
This project investigates how preprocessing pipelines, regularization strategies, and class imbalance handling affect seizure prediction performance.

## Datasets Used
- UCI Epileptic Seizure Recognition Dataset
- CHB-MIT EEG Dataset
- Bonn EEG Dataset

## Techniques
- Logistic Regression
- L1 Regularization
- L2 Regularization
- Elastic Net
- PCA
- Feature Selection
- SMOTE
- Class Weighting

## Evaluation Metrics
- Accuracy
- F1 Score
- PR-AUC

## Key Findings
- Preprocessing order affected model performance
- Elastic Net achieved strong generalization
- L1 produced sparse feature selection
- SMOTE improved minority class detection
